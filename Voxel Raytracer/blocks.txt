Chunk ID from position	
cId = (cx * worldSize + cz) * worldSize + cy

Chunk position from ID	
cy = cId % worldSize
cz = (cId / worldSize) % worldSize
cx = cId / (worldSize * worldSize)

Voxel ID from local block position	
vId = vz + chunkSize * (vy + chunkSize * vx)

Local block position from voxel ID	
vx = vId / (chunkSize * chunkSize)
vy = (vId / chunkSize) % chunkSize
vz = vId % chunkSize
